Darwin Visor: exporta datos geoespaciales sin código

Darwin Visor: export geospatial data without code

Nuestra herramienta interna en Python que exporta datos de Earth Engine y otras colecciones sin escribir código, usando tu cuenta personal y cumpliendo el uso gratuito de EE (no comercial). Construido con Earth Engine, MLflow y geemap.

Our internal tool built in Python to export Earth Engine and other collections without coding, using your personal account and honoring EE's free-use (non-commercial) policy. Built with Earth Engine, MLflow and geemap.

Esta herramienta interna tiene como objetivo simplificar la descarga y organización de colecciones satelitales y capas derivadas mediante una interfaz clara. Incluye parámetros técnicos (escala, proyección, bandas, índices) y flujos reproducibles de exportación para acelerar el trabajo personal de análisis. Incluso integra MLflow para aquellos científicos haciendo macro análisis!

This internal tool aims to streamline downloading and organizing satellite collections and derived layers through a clean UI. It captures technical parameters (scale, projection, bands, indices) and reproducible export flows to speed up personal analysis work. It even integrates MLflow for those scientists doing macro analysis!

Resultados e impacto

Outcomes & impact

Darwin Visor acelera la obtención de capas listas para análisis (índices, composiciones, mosaicos) y reduce errores al estandarizar parámetros de exportación. Diseñado para uso personal con datos públicos de EE y cumplimiento no comercial, mejora la trazabilidad y la repetibilidad del trabajo diario.

Darwin Visor speeds up getting analysis-ready layers (indices, composites, mosaics) while reducing errors via standardized export settings. Built for personal use with public EE data and non-commercial compliance, it improves traceability and repeatability.
